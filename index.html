21 Cassavage

Web App

21 Cassavage is a name we gave to our competition on kaggle and we emerged winners. We adopted the name for the web application. This is a friendly web application that uses a machine learning model to classify cassava leaf images into one of several categories to help farmers identify common diseases.

The app is integrated with a feedback system where we can receive import information from farmers to dynamically improve the performance of the model.

Question Answering System

Project

This is a Question-Answering (QA) system designed to efficiently retrieve relevant passages from a corpus of documents in response to user queries. The system utilizes SentenceTransformers for embedding generation, Elasticsearch for data storage and retrieval, Flask for API development, and Streamlit for the user interface..

Object Detection

Project & Web App

In this project, I built an object detection model using fasterrcnn and finally deployed it on streamlit. Users can upload images or use their webcam to detect and visualize objects live. The app provides an intuitive interface for adjusting detection settings and viewing results, making it a versatile tool for real-time object recognition tasks.
